I. Temat pracy:
Wykrywanie mas patologicznych w RTG klatki piersiowej z użyciem sieci neuronowych.
Detection of Pathological Masses in Chest X-rays Using Neural Networks.
II. Cel pracy:
Celem pracy jest opracowanie systemu automatycznej detekcji mas patologicznych w obrazach RTG klatki
piersiowej z wykorzystaniem wybranych bibliotek języka Python umożliwiających efektywną analizę oraz
przetwarzanie obrazów medycznych.
III. Plan pracy (zakres pracy):
● Omówienie i wybranych metod uczenia maszynowego i zadania klasyfikacji.
● Przegląd architektury sieci neuronowych stosowanych w analizie obrazów medycznych.
● Zagadnienia patologii w obrazach RTG klatki piersiowej.
● Akwizycja i przygotowanie danych z repozytorium medycznego do analizy.
● Wybór i implementacja modelu oraz realizacja zadania klasyfikacji.
● Dyskusja wyników i wnioski.
